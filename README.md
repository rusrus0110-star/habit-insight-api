# Habit Insight API

REST API for a habit analytics application built with **Node.js**, **Express**, **MongoDB**, **Mongoose** and **JWT authentication**.

The project is designed as a portfolio backend application. It includes user authentication, habit tracking, progress history, streak calculation and future analytics endpoints based on MongoDB aggregation.

---

## Project Purpose

Habit Insight API is a backend service for tracking personal habits and analyzing user progress.

The application allows users to:

- register and log in with JWT authentication
- create, read, update and delete habits
- mark habits as completed
- track daily progress
- calculate current streaks and best streaks
- store mood and notes for each habit completion
- analyze productivity patterns through statistics endpoints
